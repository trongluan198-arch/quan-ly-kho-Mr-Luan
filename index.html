🔥 Firebase — Đồng bộ nhiều máy realtime
🔥 Hướng dẫn tạo Firebase (5 phút, miễn phí):
1. Vào console.firebase.google.com → Đăng nhập Google
2. Add project → Đặt tên (vd: kho-quan-ly) → Continue → Create
3. Menu trái → Build → Realtime Database → Create Database
4. Chọn vị trí asia-southeast1 → Start in test mode → Enable
5. Tab Rules → Sửa thành ".read":true,".write":true → Publish
6. Menu trái → ⚙️ Project Settings → General → Kéo xuống Your apps → Click </> → Register app
7. Copy đoạn firebaseConfig → Dán từng field vào ô bên dưới → Kết nối

✅ Sau khi kết nối, tất cả máy mở cùng file HTML + cùng Firebase config = đồng bộ realtime tự động.

🔧 Firebase Config

🌐 Dùng nhiều máy — 3 bước đơn giản
1
Tạo Firebase miễn phí
Vào console.firebase.google.com → Add project → Realtime Database → Start in test mode
2
Dán config vào ô bên trái → Kết nối
Project Settings → General → Your apps → </> → Copy firebaseConfig
3
Copy file HTML sang các máy khác
Mỗi máy cũng nhập config → Kết nối. Tất cả sẽ đồng bộ realtime qua Firebase.
